# 5gpn block seed: encrypted-DNS / HTTPDNS bypass block (suffix match).
#
# Purpose: return NXDOMAIN for known DoH/DoT and app-HTTPDNS resolver hostnames
# so a client that tries to bypass 5gpn-dns falls back to the system resolver
# (= 5gpn-dns). This is name-based only: a resolver reached by HARDCODED IP (no
# DNS lookup) cannot be blocked here — on the intranet/peer topology that traffic
# never transits the gateway. See docs/DESIGN.md.
#
# Match type: suffix (bare = suffix). A companion block.keyword.txt carries the
# `httpdns`/`httpsdns` substring rules that catch unlisted *httpdns* endpoints.
# Tradeoff: a client in STRICT DoH/DoT mode (Android Private DNS "hostname", or a
# browser forced to a provider) fails closed (no DNS) rather than falling back.
# Sourced 2026-07 from kelee.one Block_HTTPDNS (app HTTPDNS) + curated browser
# DoH/DoT providers. Operator-editable; re-install does not clobber this file.
